# AI Assistant for Software Testing

A production-grade AI platform for QA teams to analyze bugs, generate test cases, and manage QA knowledge.

🚧 Under Development

## Tech Stack

- **Frontend**: Next.js, TypeScript, Tailwind CSS, shadcn/ui
- **Backend**: FastAPI, Pydantic, PostgreSQL
- **AI/ML**: Ollama, Hugging Face, LoRA fine-tuning, RAG
- **Infrastructure**: Docker, docker-compose
